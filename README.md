# Backend

A Node.js TypeScript application.

## Installation

```bash
pnpm install
```

## Build

```bash
pnpm run build
```

## Development

```bash
pnpm run dev
```

This starts the server in development mode with auto-restart on file changes.

## Run

```bash
pnpm start
```

## Usage

This is a Hello World app that prints "Hello, World!" to the console.
